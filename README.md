# 即刻AI品牌书网页第一版

这是一个静态网站版本，可直接本地预览或上传到静态服务器。

## 打开方式

推荐用本地服务器打开整个文件夹，例如：

```bash
python -m http.server 4177
```

然后访问：

```text
http://127.0.0.1:4177
```

也可以直接打开 `index.html`，但本地服务器方式更接近真实部署效果。

## 文件结构

- `index.html`：页面结构
- `css/`：视觉样式、动画、响应式布局
- `js/data.js`：导师、课程、案例、伙伴等集中数据
- `js/main.js`：弹窗、筛选、导航、编辑模式、粒子背景
- `assets/`：Logo、主视觉、导师、课程、案例、伙伴、联系方式素材

## 编辑模式

右上角点击“编辑模式”后：

- 页面文字可直接编辑
- 图片区域可替换本地图片
- 图片可删除并恢复为统一占位状态

当前版本的文字编辑会保存到浏览器本地缓存；图片替换用于本地预览，不会写回原始文件。

## 后续建议

- 将 `js/data.js` 中的占位合作伙伴名称替换为真实机构名称

- 若需要长期在线编辑，应接入后台或导出 JSON 配置能力
